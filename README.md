## Quick Run Notes
- **Python**: `pip install -r requirements.txt` (if provided) then run scripts/notebooks in the folder
- **C**: use `make` if a Makefile exists, otherwise compile with `gcc main.c -o program`
- **Java**: `javac *.java` then `java Main`
- **R**: open `.Rmd` files in RStudio and run/Knit
- **SQL**: open `.sql` files in your SQL tool (MySQL Workbench) and run the scripts
